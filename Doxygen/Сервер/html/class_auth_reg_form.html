Detailed Description

Класс диалогового окна для аутентификации и регистрации пользователей

Наследуется от QDialog и предоставляет функционал:

  • Авторизация пользователя
  • Регистрация нового пользователя
  • Восстановление пароля
  • Подтверждение email
